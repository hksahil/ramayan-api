End-points to use in API

1

All the Quotes

https://ramayan-api-by-sahil.herokuapp.com/quotes                         

2

Random Quotes

https://ramayan-api-by-sahil.herokuapp.com/quotes/random                        

3

Quotes by Characters

https://ramayan-api-by-sahil.herokuapp.com/quotes/<Name of ramayana character here>

Example 1 : https://ramayan-api-by-sahil.herokuapp.com/quotes/Ram
Example 2 : https://ramayan-api-by-sahil.herokuapp.com/quotes/Laxman
Example 3 : https://ramayan-api-by-sahil.herokuapp.com/quotes/Seeta



4

Quotes by ID

https://ramayan-api-by-sahil.herokuapp.com/quotes/specific/<Integer number here>

Example 1 : https://ramayan-api-by-sahil.herokuapp.com/quotes/specific/2
Example 2 : https://ramayan-api-by-sahil.herokuapp.com/quotes/specific/5
Example 3 : https://ramayan-api-by-sahil.herokuapp.com/quotes/specific/5
